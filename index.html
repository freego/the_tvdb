TheTVDB

A Ruby wrapper for TheTVDB.com API
download .ZIPdownload .TGZ

Get your TV series info with ease.

the_tvdb allows you to access TheTVDB.com with a more structured and Ruby-like way of doing things.

Installation

Add this line to your application's Gemfile:

gem 'the_tvdb'

And then execute:

$ bundle

Or install it yourself as:

$ gem install the_tvdb

Usage

If you are using this in a Rails project, create the initializer:

rails generate the_tvdb:install
